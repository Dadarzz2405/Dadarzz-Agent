# 🧠 Dadarzz Agent

Your AI-powered personal assistant.

---

## Quick Start (macOS)

1. **Double-click** `install.command`
2. If prompted "unidentified developer", right-click → **Open** → **Open**
3. Wait for installation to complete (~1 minute)
4. The app will open in your browser automatically!

A **"Dadarzz Agent"** app icon will appear on your Desktop for future launches.

---

## Requirements

- macOS (Apple Silicon M1/M2 supported)
- Internet connection (for first install only)
- A free [Groq API key](https://console.groq.com/keys)

---

## Troubleshooting

| Problem | Fix |
|---|---|
| "command not found: python3" | Run `xcode-select --install` in Terminal first |
| "unidentified developer" warning | Right-click the file → Open → Open |
| App won't start | Open Terminal, `cd` to this folder, run `bash install.command` |
| Port already in use | Close any other Dadarzz Agent windows first |

---

## Stopping the App

Close the Terminal window that opened, or press `Ctrl+C` in it.
